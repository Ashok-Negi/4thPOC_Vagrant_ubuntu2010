Steps to provision a vagrant machine
1. install vagrant from "https://developer.hashicorp.com/vagrant/downloads?product_intent=vagrant"
2. install virtualbox
3. download Sentinel LDK RTE from "https://supportportal.thalesgroup.com/csm?id=kb_search&u_related_product_names=50303b92db852e00d298728dae96199d&query=kbcat_drivers_%26_runtime_packages&_runtime_packages"
4. clone this repo to your local machine
5. run the command "vagrant up"
6. ssh into your vagrant machine using "vagrant ssh"
